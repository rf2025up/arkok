## 🆘 如果遇到问题

### 问题: 应用无法访问 (404)
**解决:**
- [ ] 等待 2-3 分钟 (Pod 可能还在初始化)
- [ ] 刷新页面 (Ctrl+F5)
- [ ] 检查 Sealos UI 中应用状态是否为 "Running"

### 问题: 页面加载但无法连接后端
**解决:**
- [ ] 打开浏览器开发者工具 (F12)
- [ ] 查看 Console 标签中的错误信息
- [ ] 检查大屏端是否显示 "未连接"
- [ ] 等待 5-10 秒，WebSocket 可能在初始化

### 问题: 数据不同步
**解决:**
- [ ] 刷新大屏端页面
- [ ] 检查浏览器控制台是否有 CORS 错误
- [ ] 验证后端公网地址是否可访问
- [ ] 等待 2-3 分钟后重试

### 问题: CORS 错误
**解决:**
- [ ] 这通常是临时的网络延迟
- [ ] 等待 30 秒后刷新页面
- [ ] 再次尝试创建学生
