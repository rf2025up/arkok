## 🎯 核心概念速查

### 项目三端架构
```
手机端 (Mobile)
  ├── 学生端界面 (index.html)
  └── 教师端界面 (bigscreen/index.html)

大屏端 (Bigscreen)
  └── 实时显示系统

后端服务 (Backend)
  ├── Express.js API
  └── PostgreSQL 数据库
```

### 部署流程简图
```
本地开发
  ↓
npm run build (生成 dist/)
  ↓
复制到 public/ 文件夹
  ↓
git push 到 GitHub
  ↓
Sealos 自动检测并部署
  ↓
公网应用更新 ✅
```

## 💡 常见命令速查

```bash
# 开发
npm install              # 安装依赖
npm run dev             # 启动开发服务
npm run build           # 编译生产版本
npm run preview         # 预览构建结果

# 部署
git add .               # 暂存变更
git commit -m "msg"     # 提交代码
git push                # 推送到GitHub（触发Sealos部署）

# 数据库
node init-db.js         # 初始化数据库
node server.js          # 启动后端服务

# 诊断
curl https://xysrxgjnpycd.sealoshzh.site/api/students  # 检查API
```
